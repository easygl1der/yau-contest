# Analysis and Differential Equations Individual

Please solve the following 6 problems.

1. Let the sequence of functions $\{ f _ { n } \} _ { n = 1 } ^ { \infty }$ in $L ^ { 2 } ( \mathbf { R } ^ { d } )$ satisfy that $\| f _ { n } \| _ { L ^ { 2 } } = 1$

(1)Show that there exists a subsequence of function $\{ f _ { n _ { j } } \} _ { j = 1 } ^ { \infty }$ such that $f _ { n _ { j } }$ converges weakly to some function $f$ in $L ^ { 2 } ( \mathbf R ^ { d } )$ , i.e.

$$
\left(f _ {n _ {j}}, g\right)\rightarrow (f, g)
$$

for all $g \in L ^ { 2 } ( \mathbf { R } ^ { d } )$ .

(2)If $f _ { n }  f$ weakly in $L ^ { 2 } ( \mathbf { R } ^ { d } )$ , and $\| f _ { n } \| _ { L ^ { 2 } } \to \| f \| _ { L ^ { 2 } }$ as $\begin{array} { r l } { \boldsymbol { \mathscr { n } } } & { { } \longrightarrow } \end{array} \infty$ . Show that $\| f _ { n } - f \| _ { L ^ { 2 } } \to 0$ as $n \to \infty$ .

2. Let $f : U \to \mathbf { C }$ be a non-constant holomorphic function where $U \subset \mathbf { C }$ is the open set containing the closure $\overline { { D } }$ of the unit disk $D = \{ z \in \mathbf { C } | | z | < 1 \}$ .

If $| f ( z ) | = 1$ , for all $z \in \partial D$ , Prove that $D \subset f ( { \overline { { D } } } )$ .

3. Prove that if a sequence of harmonic function on the open disk converges uniformly on compact subset of the disk, then the limit is harmonic.

4. Let $\mu$ be a Borel measure on $\mathbf { R } ^ { n }$ . Let $\rho > 0$ , a fixed positive number, and $B _ { \rho } ( x ) =$ $\{ y \in \mathbf { R } ^ { n } | d ( x , y ) < \rho \}$ . For $\boldsymbol { x } \in \mathbf { R } ^ { n }$ , define a function:

$$
\theta (x): x \rightarrow \mu (\overline {{B _ {\rho} (x)}})
$$

1) Show that $\theta$ is upper semi-continuous, i.e. for every $x \in \mathbf { R } ^ { n } , \theta ( x ) \geq \operatorname* { l i m } \operatorname* { s u p } _ { y \to x } \theta ( y )$   
2) Give an example of a Borel measure $\mu$ , such that the function $\theta$ is not continuous.

5. Let $g$ denote a smooth function on $\mathbf { R } ^ { n }$ with compact support. Let $f$ denote the function given by the formula

$$
f (x) = \frac {1}{n (n - 1) \alpha (n)} \int_ {\mathbf {R} ^ {n}} \frac {1}{| x - y | ^ {n - 2}} g (y) d y.
$$

Here $\alpha ( n )$ is volume of the unit ball in $\mathbf { R } ^ { n }$

$( a )$ Prove that the integral that defines $f$ converges for each $\boldsymbol { x } \in \mathbf { R } ^ { n }$   
$( b )$ Prove that $f$ is differentiable and that the gradient of $f$ if given by the formula

$$
\nabla f | _ {x} = \frac {1}{n (n - 1) \alpha (n)} \int_ {\mathbf {R} ^ {n}} \frac {1}{| x - y | ^ {n - 2}} (\nabla g) | _ {y} d y.
$$

(c) Prove the $f$ obeys $- \Delta f = g$ with $\Delta$ denoting $\frac { \partial ^ { 2 } } { { \partial } x _ { 1 } { } ^ { 2 } } + \cdots + \frac { \partial ^ { 2 } } { { \partial } x _ { n } { } ^ { 2 } }$

6. If $u$ is a positive harmonic function on $\mathbb { R } ^ { n } \setminus \{ 0 \}$ ( $n \geq 2$ ), then exist constants $a \ge 0 , b \ge 0$ such that

$$
u (x) = a + b | x | ^ {2 - n}
$$

for all $x \in \mathbb { R } ^ { n } \setminus \{ 0 \}$
