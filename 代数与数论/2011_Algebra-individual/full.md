1. Let $K = \mathbb { Q } ( { \sqrt { - 3 } } )$ , an imaginary quadratic field.

(a) Does there exists a finite Galois extension $L / \mathbb { Q }$ which contains $K$ such that $\operatorname { G a l } ( L / \mathbb { Q } ) \cong S _ { 3 } { } ^ { * }$ ? (Here $S _ { 3 }$ is the symmetric group in 3 letters.)   
(b) Does there exists a finite Galois extension $L / \mathbb { Q }$ which contains $K$ such that $\operatorname { G a l } ( L / \mathbb { Q } ) \cong \mathbb { Z } / 4 \mathbb { Z } ^ { \mathfrak { \cdot } }$ ?   
(c) Does there exists a finite Galois extension $L / \mathbb { Q }$ which contains $K$ such that $\operatorname { G a l } ( L / \mathbb { Q } ) \cong Q \mathrm { : }$ ? Here $Q$ is the quaternion group with 8 elements $\{ \pm 1 , \pm i , \pm j , \pm k \}$ , a finite subgroup of the group of units $\mathbb { H } ^ { \times }$ of the ring $\mathbb { H }$ of all Hamiltonian quaternions.

3. Let $F \subset \mathbb { R }$ be the subset of all real numbers that are roots of monic polynomials $f ( X ) \in \mathbb { Q } [ X ]$ .

(1) Show that $F$ is a field.   
(2) Show that the only field automorphisms of $F$ are the identity automorphism $\alpha ( x ) = x$ for all $x \in F$ .

4. Let $V$ be a finite-dimensional vector space over $\mathbb { R }$ and $T : V  V$ be a linear transformation such that

(1) the minimal polynomial of $T$ is irreducible;   
(2) there exists a vector $v \in V$ such that $\{ T ^ { \ i } v \mid i \geq 0 \}$ spans $V$

Show that $V$ contains no non-trivial proper $T$ -invariant subspace.
