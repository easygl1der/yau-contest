Problem 1. Let the special linear group (of order 2)

$$
\operatorname {S L} _ {2} (\mathbb {R}) = \{g = \left( \begin{array}{c c} a & b \\ c & d \end{array} \right) \in M _ {2} (\mathbb {R}): \det  g = 1 \}
$$

act on the upper half plane $\mathbb { H } = \left\{ z = x + i y \in \mathbb { C } : y > 0 \right\}$ linear fractionally:

$$
\left( \begin{array}{c c} a & b \\ c & d \end{array} \right) z = \frac {a z + b}{c z + d}.
$$

(a) (5 points) Prove that the action is transitive, i.e., for any two $z _ { 1 } , z _ { 2 } \in \mathbb { H }$ , there is $g \in \mathrm { S L _ { 2 } } ( \mathbb { R } )$ such that $y z _ { 1 } = z _ { 2 }$ .

(b) (5 points) For a fixed $z \in \mathbb { H }$ , prove that its stabilizer $G _ { z } = \{ g \in \mathrm { S L } _ { 2 } ( \mathbb { R } ) : g z = z \}$ is isomorphic to $\operatorname { S O } _ { 2 } ( \mathbb { R } ) = \{ g \in M _ { 2 } ( \mathbb { R } ) : g g ^ { t } = 1 \}$ , where $g ^ { t }$ is the transpose of $g$ .

(c)(10 points) Let $\mathbb { Z }$ be the set of integers and let

$$
\Gamma (2) = \left\{\left( \begin{array}{c c} a & b \\ c & d \end{array} \right) \in \operatorname {S L} _ {2} (\mathbb {R}):   a, b, c, d \in \mathbb {Z}, \quad a - 1 \equiv d - 1 \equiv b \equiv c \equiv 0 \pmod {2} \right\}
$$

be a discrete subgroup of $\operatorname { S L _ { 2 } } ( \mathbb { R } )$ (no need to prove this), and let it act on $\mathbb { Q } \cup \lbrace \infty \rbrace$ linearly fractionally as above. How many orbits does this action have? Give a representative for each orbit.
