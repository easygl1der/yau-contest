\section{2021 Analysis \& Differential Equations}

\subsection{Exam}

\begin{exercise}
Prove that $f(x)\equiv 0$ is the only solution in $L^2(\mathbb{R}^n)$ such that
\[
\triangle f = 0.
\]
\end{exercise}

\begin{solution}
\textbf{Prerequisites:}
\begin{itemize}
    \item \textbf{Fourier Transform of Derivatives:} The Fourier transform $\mathcal{F}$ converts differentiation into multiplication. Specifically, $\mathcal{F}(\triangle f)(\xi) = -|\xi|^2 \hat{f}(\xi)$, where $\triangle$ is the Laplacian operator.
    \item \textbf{Plancherel's Theorem:} A function $f$ is in $L^2(\mathbb{R}^n)$ if and only if its Fourier transform $\hat{f}$ is also in $L^2(\mathbb{R}^n)$. This means the "energy" of the function is preserved in the frequency domain.
    \item \textbf{Distributions Supported at a Point:} A tempered distribution whose support is a single point $\{0\}$ must be a finite linear combination of the Dirac delta function $\delta$ and its partial derivatives.
\end{itemize}

\textbf{Steps:}\\[2ex]
1. We start with the equation $\triangle f = 0$. Since $f \in L^2(\mathbb{R}^n)$, it is also a tempered distribution, so we can apply the Fourier transform to both sides.\\[2ex]
2. Applying the transform gives us $-|\xi|^2 \hat{f}(\xi) = 0$. This equation holds in the sense of distributions. It tells us that the support of the distribution $\hat{f}(\xi)$ must be contained within the set where $|\xi|^2 = 0$, which is just the origin $\{0\}$.\\[2ex]
3. Because the support of $\hat{f}$ is contained in $\{0\}$, $\hat{f}$ must be a linear combination of the Dirac delta function and its derivatives.\\[2ex]
4. Now we apply the $L^2$ condition. Plancherel's theorem states that $\hat{f}$ must be an $L^2$ function. However, the Dirac delta function and its derivatives are highly singular distributions and do not belong to $L^2(\mathbb{R}^n)$.\\[2ex]
5. The only distribution supported at $\{0\}$ that is actually a function in $L^2$ is the zero function. Thus, $\hat{f} = 0$ in $L^2(\mathbb{R}^n)$.\\[2ex]
6. Since the Fourier transform is an isomorphism on $L^2$, $\hat{f} = 0$ implies that $f = 0$ in $L^2$. Since harmonic functions are smooth, $f \equiv 0$ everywhere.

\textbf{Intuition:}
Harmonic functions (those satisfying $\triangle f = 0$) satisfy the "mean value property," meaning the value at any point is the average of its neighbors. In a bounded domain, you can have many such functions. However, in the infinite space $\mathbb{R}^n$, the requirement that the function's square is integrable ($L^2$) is very strict. It essentially says the function must "die out" at infinity. A non-constant harmonic function simply cannot decay fast enough to be in $L^2$ without just being zero everywhere.
\end{solution}

\begin{exercise}
Let $X\subset C([0,1])$ be a finite dimensional linear subspace of the space of real-valued continuous functions on $[0,1]$. Show that, for a sequence of functions $\{f_k\}_{k\geq 1}\subset X$, if it converges pointwise, it converges uniformly.
\end{exercise}

\begin{solution}
\textbf{Prerequisites:}
\begin{itemize}
    \item \textbf{Basis of a Vector Space:} Any function in an $n$-dimensional space $X$ can be uniquely written as $f(x) = \sum_{j=1}^n \alpha_j \varphi_j(x)$, where $\{\varphi_1, \dots, \varphi_n\}$ is a fixed basis.
    \item \textbf{Evaluation Functionals:} For any point $t \in [0,1]$, the map $\ell_t(f) = f(t)$ is a linear functional on $X$.
    \item \textbf{Linear Independence of Evaluation Maps:} If $X$ is $n$-dimensional, we can find $n$ distinct points $x_1, \dots, x_n$ such that knowing the values $f(x_1), \dots, f(x_n)$ completely determines the function $f$.
\end{itemize}

\textbf{Steps:}\\[2ex]
1. Let $n = \dim X$ and pick a basis $\{\varphi_1, \dots, \varphi_n\}$ for $X$.\\[2ex]
2. Consider the evaluation functionals $\ell_t: X \to \mathbb{R}$ given by $\ell_t(f) = f(t)$. Since a function in $X$ that is zero everywhere must be the zero function, the intersection of the kernels of all these functionals is trivial: $\bigcap_{t \in [0,1]} \ker(\ell_t) = \{0\}$.\\[2ex]
3. Because $X$ is finite-dimensional, there must exist a finite set of $n$ points $x_1, x_2, \dots, x_n \in [0,1]$ such that $\bigcap_{i=1}^n \ker(\ell_{x_i}) = \{0\}$. This means the evaluation matrix $A = (\varphi_j(x_i))$ has a non-zero determinant and is invertible.\\[2ex]
4. We can write each function $f_k$ in terms of our basis as $f_k(x) = \sum_{j=1}^n \alpha_j^{(k)} \varphi_j(x)$.\\[2ex]
5. Evaluating this at our chosen points gives a system of equations: the vector of values $(f_k(x_1), \dots, f_k(x_n))^T$ is equal to the matrix $A$ multiplied by the coefficient vector $(\alpha_1^{(k)}, \dots, \alpha_n^{(k)})^T$.\\[2ex]
6. Since $A$ is invertible, we can multiply by $A^{-1}$ to express the coefficients $\alpha_j^{(k)}$ directly in terms of the point evaluations $f_k(x_i)$.\\[2ex]
7. We are given that $\{f_k\}$ converges pointwise, so it converges at the points $x_1, \dots, x_n$. Therefore, the vector of evaluations converges, which implies the coefficient vector $(\alpha_1^{(k)}, \dots, \alpha_n^{(k)})^T$ converges to some limit $(\alpha_1, \dots, \alpha_n)^T$.\\[2ex]
8. The sequence $\{f_k\}$ thus converges uniformly to the function $\alpha_1 \varphi_1 + \dots + \alpha_n \varphi_n$.

\textbf{Intuition:}
Think of a finite-dimensional space of functions as being very "stiff" or "rigid." For example, a straight line is a 2D space of functions ($f(x) = ax+b$), which is perfectly determined by just two points. Because our space $X$ is finite-dimensional, a specific finite number of points entirely "pins down" the whole function. If a sequence of functions converges at these pinning points, their entire underlying structure (their coefficients) must stabilize, forcing convergence everywhere uniformly.
\end{solution}

\begin{exercise}
\begin{enumerate}
\item[(a)] For $f\in L^1(\mathbb{R}^n)$, $g\in L^{\infty}(\mathbb{R}^n)$, show that their convolution $f*g$ is a well-defined continuous function.
\item[(b)] Let $E\subset\mathbb{R}^n$ be a Lebesgue measurable set with Lebesgue measure $m(E)>0$. Prove that
\[
E-E := \{x-y\mid x\in E, y\in E\}
\]
contains an open neighborhood of $\mathbf{0}\in\mathbb{R}^n$.
\end{enumerate}
\end{exercise}

\begin{solution}
\textbf{Prerequisites:}
\begin{itemize}
    \item \textbf{Hölder's Inequality:} For convolution, $|(f*g)(x)| \leq \|f\|_1 \|g\|_\infty$. This ensures the integral doesn't "blow up" and is well-defined.
    \item \textbf{Continuity of Translation in $L^1$:} For any $f \in L^1$, the "shifted" version of $f$ gets closer to $f$ in the $L^1$ sense as the shift $x \to 0$. Formally, $\lim_{x \to 0} \int |f(x_0+x-y) - f(x_0-y)| dy = 0$.
    \item \textbf{Indicator Functions:} The function $\mathbf{1}_E(x)$ is $1$ if $x \in E$ and $0$ otherwise. The convolution of indicator functions can be used to measure the overlap of translated sets.
\end{itemize}

\textbf{Steps:}\\[2ex]
\textbf{Part (a):}\\[2ex]
1. Define the convolution $(f*g)(x) = \int_{\mathbb{R}^n} f(x-y)g(y)dy$. By Hölder's inequality, we have $\|f*g\|_\infty \leq \|f\|_1 \|g\|_\infty < \infty$, so it is well-defined.\\[2ex]
2. To prove continuity, we examine the difference $|(f*g)(x_0+x) - (f*g)(x_0)|$.\\[2ex]
3. Writing out the integrals, we can bound this difference:
   \[
   \left| \int_{\mathbb{R}^n} (f(x_0+x-y) - f(x_0-y))g(y) dy \right| \leq \|g\|_\infty \int_{\mathbb{R}^n} |f(x_0+x-y) - f(x_0-y)| dy
   \]
4. By the continuity of translation in $L^1$, as $x \to 0$, the integral of the shifted difference converges to $0$. Thus, $f*g$ is continuous.\\[2ex]

\textbf{Part (b):}\\[2ex]
1. It suffices to consider the case where $m(E) < \infty$ (if it's infinite, just take a subset of finite positive measure).\\[2ex]
2. Let $f = \mathbf{1}_E$ and $g = \mathbf{1}_{-E}$. Since $m(E) < \infty$, $f \in L^1(\mathbb{R}^n)$ and clearly $g \in L^\infty(\mathbb{R}^n)$.\\[2ex]
3. From part (a), the function $h(x) = (f * g)(x)$ is a continuous function.\\[2ex]
4. Let's evaluate $h$ at $x=0$: $h(0) = \int \mathbf{1}_E(-y)\mathbf{1}_{-E}(y) dy = \int \mathbf{1}_E(y) dy = m(E)$, which is strictly greater than $0$.\\[2ex]
5. Because $h$ is continuous and $h(0) > 0$, there must exist an open set $U$ around the origin and a constant $\delta > 0$ such that $h(x) > \delta > 0$ for all $x \in U$.\\[2ex]
6. For any $x \in U$, by definition $h(x) = \int \mathbf{1}_E(x-y)\mathbf{1}_{-E}(y) dy > 0$. This implies the integrand is positive for some $y$.\\[2ex]
7. Therefore, there must be some $y \in -E$ such that $x-y \in E$. Since $y \in -E$, we can write $y = -y'$ for some $y' \in E$. Thus $x + y' \in E$, meaning $x \in E - y'$.\\[2ex]
8. This implies $x \in E - E$. Hence, the open neighborhood $U$ is entirely contained within $E-E$.

\textbf{Intuition:}
(a) Convolution is a "smearing" process. When you take a rough function $g$ and smear it with $f$, the output inherits the smooth translation properties of $f$. Even if $g$ is a step function, the "averaging" effect makes the output continuous.\\[2ex]
(b) This is known as Steinhaus's Theorem. It says that if a set has a positive size, it can't be "infinitely scattered" everywhere without any bulk. If you shift the set by a tiny microscopic amount, it \textit{must} still overlap with its original self. That overlap proves that the tiny shift can be expressed as the difference between two points within the set.
\end{solution}

\begin{exercise}
Assume that $P$ is a polynomial with complex coefficients. Prove that there exists infinitely many solutions of the following equations on $\mathbb{C}$:
\[
e^z = P(z).
\]
\end{exercise}

\begin{solution}
\textbf{Prerequisites:}
\begin{itemize}
    \item \textbf{Essential Singularity at Infinity:} A function $F(z)$ has an essential singularity at $z = \infty$ if $F(1/z)$ has an essential singularity at $z = 0$. For instance, $e^z$ fluctuates wildly as $z \to \infty$.
    \item \textbf{Big Picard's Theorem:} If a complex function has an essential singularity at a point, then in any neighborhood of that singularity, the function takes on all possible complex values, with at most one exception, infinitely often.
\end{itemize}

\textbf{Steps:}\\[2ex]
1. Let's rewrite the equation as $F(z) = \frac{e^z}{P(z)} = 1$. We want to show that $F(z) = 1$ has infinitely many solutions.\\[2ex]
2. Consider the behavior of $F(z)$ at $z = \infty$. The exponential function $e^z$ has an essential singularity at $\infty$, while $P(z)$ is a polynomial and has a pole at $\infty$.\\[2ex]
3. The quotient of a function with an essential singularity and a function with a pole still retains an essential singularity. Thus, $F(z)$ has an essential singularity at $z = \infty$.\\[2ex]
4. We now apply Big Picard's Theorem at $\infty$. The theorem states that in any neighborhood of infinity (which means the region $|z| > R$ for any arbitrarily large $R$), $F(z)$ will take every complex value infinitely many times, with at most one possible exception.\\[2ex]
5. Let's check the value $0$. Since $e^z \neq 0$ for all $z \in \mathbb{C}$, the function $F(z) = \frac{e^z}{P(z)}$ never takes the value $0$. So $0$ is the one and only allowed exceptional value.\\[2ex]
6. Because $0$ is the unique exception, $F(z)$ \textit{must} take the value $1$ infinitely many times.\\[2ex]
7. Therefore, the equation $e^z = P(z)$ has infinitely many solutions in $\mathbb{C}$.

\textbf{Intuition:}
Polynomials are relatively "tame" functions—they grow predictably as you move away from the origin. The exponential function $e^z$, however, is incredibly "wild" near infinity; it can grow massively, shrink to zero, or spiral around the complex plane depending on the direction you travel. When you pit $e^z$ against $P(z)$, the wild behavior of the exponential completely dominates. Big Picard's Theorem guarantees that this wild oscillation forces the exponential to cross paths with the polynomial infinitely many times.
\end{solution}

\begin{exercise}
Let $f$ be a bounded holomorphic function defined on $B=\{z\mid 0<\operatorname{Re}(z)<1\}$ that can be extended as a continuous function on $\bar{B}$. Let
\[
A_0 = \sup_{\operatorname{Re}(z)=0}|f(z)| > 0,\quad A_1 = \sup_{\operatorname{Re}(z)=1}|f(z)| > 0.
\]
Prove that for all $z\in B$, we have
\[
|f(z)| \leq (A_0)^{1-\operatorname{Re}(z)}(A_1)^{\operatorname{Re}(z)}.
\]
\end{exercise}

\begin{solution}
\textbf{Prerequisites:}
\begin{itemize}
    \item \textbf{Maximum Modulus Principle:} If a holomorphic function is defined on a bounded region, its maximum absolute value must occur on the boundary of that region.
    \item \textbf{Phragmén-Lindelöf Principle:} For unbounded regions like an infinite strip, the Maximum Modulus Principle can be rescued by multiplying the function by a small "decay factor" (like $e^{\varepsilon z^2}$) that forces the function to zero far away, allowing us to safely apply the principle on a bounded subset.
\end{itemize}

\textbf{Steps:}\\[2ex]
1. We construct an auxiliary function $g(z) = f(z) (A_0)^{z-1} (A_1)^{-z}$. This is a holomorphic function defined on $B$.\\[2ex]
2. The purpose of this function is to "flatten" the boundary values. Let's check its magnitude on the boundaries. On the line $\operatorname{Re}(z) = 0$, $|g(iy)| = |f(iy)| A_0^{-1} A_1^0 \leq A_0 \cdot A_0^{-1} = 1$.\\[2ex]
3. On the line $\operatorname{Re}(z) = 1$, $|g(1+iy)| = |f(1+iy)| A_0^0 A_1^{-1} \leq A_1 \cdot A_1^{-1} = 1$. So $g(z)$ is bounded by $1$ on both edges of the strip.\\[2ex]
4. However, the strip is unbounded vertically ($y \to \pm\infty$). To apply the Maximum Modulus Principle, we introduce a decay factor and consider a new function $h(z) = g(z) e^{\varepsilon z^2}$ for some small $\varepsilon > 0$.\\[2ex]
5. Notice that $|e^{\varepsilon z^2}| = |e^{\varepsilon (x^2 - y^2 + 2ixy)}| = e^{\varepsilon(x^2 - y^2)}$. Because $x \in [0,1]$, as $y \to \pm\infty$, the term $-y^2$ dominates, and the factor $e^{\varepsilon(x^2 - y^2)}$ rapidly goes to $0$.\\[2ex]
6. Because $h(z)$ goes to $0$ as $z \to \pm i\infty$, it is bounded. We can draw a large rectangle, and on the top and bottom edges, $|h(z)|$ is extremely small. On the left and right edges, $|h(z)| \leq 1 \cdot e^{\varepsilon x^2} \leq e^\varepsilon$.\\[2ex]
7. By the Maximum Modulus Principle, $|h(z)|$ is bounded by its maximal value on the boundary. Letting $\varepsilon \to 0$, the $e^{\varepsilon z^2}$ factor disappears, and we conclude $|g(z)| \leq 1$ for all $z \in B$.\\[2ex]
8. Expanding $|g(z)| \leq 1$ gives $|f(z) (A_0)^{z-1} (A_1)^{-z}| \leq 1$. Taking the absolute value gives $|f(z)| A_0^{x-1} A_1^{-x} \leq 1$, where $x = \operatorname{Re}(z)$.\\[2ex]
9. Rearranging this inequality yields $|f(z)| \leq (A_0)^{1-\operatorname{Re}(z)} (A_1)^{\operatorname{Re}(z)}$, which completes the proof.

\textbf{Intuition:}
Imagine the complex strip as a bridge stretched between two infinitely long walls. The walls have heights bounded by $A_0$ and $A_1$. The Phragmén-Lindelöf trick shows that a holomorphic function cannot "spike" in the middle of the bridge if it remains calm at the boundaries and doesn't explode too fast at infinity. Instead, its interior height is constrained by a weighted geometric average of the boundary heights, where the weight depends exactly on your horizontal distance to each wall.
\end{solution}

\begin{exercise}
Assume that $\Omega\subset\mathbb{R}^n$ is a bounded domain with smooth boundary. Prove that there exists a positive constant $\varepsilon_0$ so that for all real numbers $\varepsilon<\varepsilon_0$, for all $f\in L^2(\Omega)$, there exists a unique $u\in H_0^1(\Omega)$ so that
\[
-\triangle u + \varepsilon\sin(u) = f
\]
in the sense of distributions.
\end{exercise}

\begin{solution}
\textbf{Prerequisites:}
\begin{itemize}
    \item \textbf{Direct Method of Calculus of Variations:} A powerful technique to solve PDEs by reframing them as optimization problems. We construct an "energy" functional; the function that minimizes this energy is the solution to the PDE.
    \item \textbf{Poincaré Inequality:} For functions $u \in H_0^1(\Omega)$ (functions that vanish on the boundary), the $L^2$ norm of the function is controlled by the $L^2$ norm of its gradient: $\|u\|_{L^2}^2 \leq \frac{1}{\lambda_1(\Omega)} \|\nabla u\|_{L^2}^2$, where $\lambda_1(\Omega)$ is a positive constant depending only on the domain.
\end{itemize}

\textbf{Steps:}\\[2ex]
1. \textbf{Existence:} We reframe the PDE as a minimization problem. Consider the energy functional defined on $H_0^1(\Omega)$:
   \[
   E(u) = \int_\Omega \left( \frac{1}{2}|\nabla u|^2 - \varepsilon\cos(u) - fu \right) dx
   \]
   A minimizer of this functional will satisfy the Euler-Lagrange equation $-\triangle u + \varepsilon\sin(u) = f$.\\[2ex]
2. We must show $E(u)$ is bounded below. Notice that $-\varepsilon\cos(u)$ is bounded. By Cauchy-Schwarz and the Poincaré inequality, the linear term $-fu$ grows slower than the quadratic term $\frac{1}{2}|\nabla u|^2$. Thus, the functional is bounded below and coercive. Therefore, a minimizing sequence converges weakly to a minimizer $u \in H_0^1(\Omega)$, proving existence.\\[2ex]
3. \textbf{Uniqueness:} Suppose there are two solutions, $u_1$ and $u_2$, in $H_0^1(\Omega)$. They both satisfy the PDE, so their difference satisfies:
   \[
   -\triangle(u_1 - u_2) + \varepsilon(\sin(u_1) - \sin(u_2)) = 0
   \]
4. Multiply this equation by $(u_1 - u_2)$ and integrate by parts over the domain $\Omega$:
   \[
   \int_\Omega |\nabla(u_1 - u_2)|^2 dx + \varepsilon \int_\Omega (u_1 - u_2)(\sin(u_1) - \sin(u_2)) dx = 0
   \]
5. Rearranging this leads to the inequality:
   \[
   \|\nabla(u_1 - u_2)\|_{L^2}^2 = \varepsilon \int_\Omega (u_1 - u_2)(\sin(u_2) - \sin(u_1)) dx \leq \varepsilon \int_\Omega |u_1 - u_2||\sin(u_1) - \sin(u_2)| dx
   \]
6. Using the Lipschitz property of the sine function ($|\sin(u_1) - \sin(u_2)| \leq |u_1 - u_2|$), we get:
   \[
   \|\nabla(u_1 - u_2)\|_{L^2}^2 \leq \varepsilon \int_\Omega |u_1 - u_2|^2 dx = \varepsilon \|u_1 - u_2\|_{L^2}^2
   \]
7. Now, apply the Poincaré inequality to the right-hand side, $\|u_1 - u_2\|_{L^2}^2 \leq \frac{1}{\lambda_1(\Omega)} \|\nabla(u_1 - u_2)\|_{L^2}^2$:
   \[
   \|\nabla(u_1 - u_2)\|_{L^2}^2 \leq \frac{\varepsilon}{\lambda_1(\Omega)} \|\nabla(u_1 - u_2)\|_{L^2}^2
   \]
8. If we set $\varepsilon_0 = \lambda_1(\Omega)$, then for any $\varepsilon < \varepsilon_0$, the multiplier $\frac{\varepsilon}{\lambda_1(\Omega)}$ is strictly less than $1$. The only way a number can be strictly less than itself multiplied by a fraction is if the number is zero. Thus, $\|\nabla(u_1 - u_2)\|_{L^2}^2 = 0$, which means $u_1 = u_2$.

\textbf{Intuition:}
This PDE models a physical system settling into its lowest energy state. The "kinetic energy" ($\frac{1}{2}|\nabla u|^2$) pushes the system toward a smooth, flat state, while the non-linear term ($\varepsilon\cos(u)$) acts as a small, rippled potential landscape trying to trap it. Because the kinetic energy is dominant, the system is guaranteed to have a stable bottom (existence). If the ripples are "weak" enough (meaning $\varepsilon$ is small), they aren't deep enough to create multiple isolated valleys. The dominant kinetic energy ensures the landscape is strictly bowl-shaped overall, guaranteeing there is only one unique bottom.
\end{solution}
