\begin{exercise}
Let $(M_n,\|\cdot\|)$ be the Banach space formed by $n\times n$ real matrices, equipped with Hilbert-Schmidt norm. Prove:

\begin{itemize}
\item[(1)] If $\gamma:\mathbb{R}\to M_n$ is a $C^1$-function with $\gamma(0)=\mathbb{I}$ (the identity matrix) and $\dot{\gamma}(0)=A\in M_n$, then for $\forall t\in\mathbb{R}$, the sequence $\{\gamma^n(t/n)\}_{n=1}^{\infty}$ converges to $\exp(tA)$. In particular, for $\forall A,B\in M_n$, $e^{t(A+B)} = \lim_{n\to\infty}(e^{\frac{t}{n}A}e^{\frac{t}{n}B})^n$.
\item[(2)] For $\forall A,B\in M_n$, if $e^{t(A+B)} = e^{tA}e^{tB}$, $\forall t\in\mathbb{R}$, then $[A,B]:=AB-BA=0$.
\end{itemize}
\end{exercise}

\begin{solution}
\textbf{Motivation:} Part (1) is the Lie-Trotter Product Formula, which is fundamental in quantum mechanics (operator splitting) and numerical analysis. It shows how the exponential of a sum can be approximated by products of exponentials. Part (2) shows that the commutativity of the exponentials for all $t$ implies the commutativity of the generators.

(1) Since $\gamma$ is $C^1$, we have the Taylor expansion of $\gamma$ at $h=0$:
\[ \gamma(h) = \gamma(0) + h \dot{\gamma}(0) + o(h) = I + hA + o(h) \]
as $h \to 0$. Substituting $h = t/n$, we get:
\[ \gamma(t/n) = I + \frac{t}{n}A + o(t/n) \]
as $n \to \infty$. To find the limit of $\gamma^n(t/n)$, we take the logarithm (for sufficiently large $n$, $\gamma(t/n)$ is near $I$):
\[ n \ln(\gamma(t/n)) = n \ln\left(I + \frac{tA}{n} + o(1/n)\right) = n \left( \frac{tA}{n} + o(1/n) \right) = tA + o(1) \]
Thus, $\lim_{n\to\infty} \gamma^n(t/n) = \exp(\lim_{n\to\infty} n \ln(\gamma(t/n))) = \exp(tA)$.
For the second part, let $\gamma(t) = e^{tA} e^{tB}$. Then $\gamma(0) = I$ and $\dot{\gamma}(0) = A e^0 e^0 + e^0 B e^0 = A+B$. Applying the first part gives the Trotter Product Formula: $e^{t(A+B)} = \lim_{n\to\infty} (e^{\frac{t}{n}A} e^{\frac{t}{n}B})^n$.

(2) Suppose $e^{t(A+B)} = e^{tA} e^{tB}$ for all $t$. Differentiating both sides with respect to $t$:
LHS: $\frac{d}{dt} e^{t(A+B)} = (A+B) e^{t(A+B)}$.
RHS: $\frac{d}{dt} (e^{tA} e^{tB}) = A e^{tA} e^{tB} + e^{tA} B e^{tB}$.
So, $(A+B) e^{t(A+B)} = A e^{t(A+B)} + e^{tA} B e^{tB}$, which simplifies to $B e^{t(A+B)} = e^{tA} B e^{tB}$.
Multiplying by $e^{-tA}$ on the left, we have $e^{-tA} B e^{t(A+B)} = B e^{tB}$.
Differentiate this identity with respect to $t$ again and evaluate at $t=0$:
LHS: $\frac{d}{dt}\big|_{t=0} (e^{-tA} B e^{t(A+B)}) = -A B e^0 + B (A+B) e^0 = -AB + BA + B^2 = [B, A] + B^2$.
RHS: $\frac{d}{dt}\big|_{t=0} (B e^{tB}) = B^2 e^0 = B^2$.
Equating the results, we get $[B, A] + B^2 = B^2$, which implies $[B, A] = 0$, hence $[A, B] = 0$.
\end{solution}

\begin{exercise}
Let a second-order ordinary differential equation be given:
\[
\frac{d^2 x}{dt^2} = f(t,x,\frac{d}{dt}x), \quad (*)
\]
where $f(t,x,y)$ is a $C^1$ function on $[0,1]\times\mathbb{R}^2$. Let $\varphi$, $t\in[0,1]$ be a solution of the second-order such that $\varphi(0)=a$, $\varphi(1)=b$ where $a,b$ are given real numbers. Suppose $\frac{\partial f}{\partial x}(t,x,y) > 0$ for $(t,x,y)\in[0,1]\times\mathbb{R}^2$. Prove: if $|\beta-b|$ is sufficiently small, $\beta\in\mathbb{R}$, then there exists a solution $\psi$ of $(*)$ such that $\psi(0)=a$, $\psi(1)=\beta$.
\end{exercise}

\begin{solution}
\textbf{Motivation:} This problem is about the existence of solutions to boundary value problems (BVPs) through the shooting method. The condition $\partial f / \partial x > 0$ ensures that solutions to the IVP are monotonic with respect to their initial slope, preventing the existence of "conjugate points" and ensuring the invertibility of the shooting map.

Consider the initial value problem (IVP): $\frac{d^2 x}{dt^2} = f(t,x, \dot{x})$ with $x(0) = a$ and $\dot{x}(0) = \xi$. Let $x(t, \xi)$ be the unique solution of this IVP. We are given that for $\xi_0 = \dot{\varphi}(0)$, the solution $x(t, \xi_0) = \varphi(t)$ satisfies $x(1, \xi_0) = b$.
To show existence of solutions for $\beta$ near $b$, we use the Implicit Function Theorem on the map $G(\xi) = x(1, \xi)$. It suffices to show $\frac{\partial x}{\partial \xi}(1, \xi_0) \neq 0$.
Let $z(t) = \frac{\partial x}{\partial \xi}(t, \xi_0)$. Then $z(t)$ satisfies the linearized (variational) equation:
\[ \ddot{z}(t) = \frac{\partial f}{\partial x}(t, \varphi, \dot{\varphi}) z(t) + \frac{\partial f}{\partial y}(t, \varphi, \dot{\varphi}) \dot{z}(t) \]
with initial conditions $z(0) = \frac{\partial x}{\partial \xi}(0, \xi_0) = 0$ and $\dot{z}(0) = \frac{\partial \dot{x}}{\partial \xi}(0, \xi_0) = 1$.
Suppose $z(1) = 0$. Since $z(0)=0$ and $\dot{z}(0)=1$, $z(t) > 0$ for small $t > 0$. If $z(1)=0$, then $z(t)$ must have a local maximum at some $t_0 \in (0, 1)$ where $z(t_0) > 0$.
At this maximum, we have $\dot{z}(t_0) = 0$ and $\ddot{z}(t_0) \leq 0$.
However, the variational equation gives:
\[ \ddot{z}(t_0) = \frac{\partial f}{\partial x}(t_0, \dots) z(t_0) + \frac{\partial f}{\partial y}(t_0, \dots) \cdot 0 = \frac{\partial f}{\partial x}(t_0, \dots) z(t_0) \]
Since $\frac{\partial f}{\partial x} > 0$ and $z(t_0) > 0$, we have $\ddot{z}(t_0) > 0$. This is a contradiction.
Thus $z(t)$ cannot have a positive maximum, and since it starts positive, it can never return to 0. Hence $z(1) \neq 0$.
By the Inverse Function Theorem, $G(\xi)$ is a local diffeomorphism near $\xi_0$, so for $\beta$ sufficiently close to $b$, there exists $\xi$ such that $x(1, \xi) = \beta$.
\end{solution}

\begin{exercise}
A function $\varphi:\mathbb{R}^n\to\mathbb{C}$ is called positive definite if for all $k\in\mathbb{N}$, $y_j\in\mathbb{R}^n$, $c_j\in\mathbb{C}$, $j=1,\dots,k$, one has $\sum_{i,j=1}^k c_i\bar{c}_j\varphi(y_i-y_j) \geq 0$. If $\varphi$ is a measurable positive definite function on $\mathbb{R}^n$. Prove:

\begin{itemize}
\item[(1)] $\varphi(-y) = \overline{\varphi(y)}$ and $|\varphi(y)|\leq\varphi(0)$.
\item[(2)] For every Lebesgue integrable nonnegative function $f$ on $\mathbb{R}^n$, one has
\[
\int_{\mathbb{R}^n}\varphi(x-y)f(x)f(y)\,dx\,dy \geq 0.
\]
\item[(3)] If the function $f$ is also even, then
\[
\int_{\mathbb{R}^n}\varphi(x)f*f(x)\,dx \geq 0,
\]
where $*$ stands for convolution.
\item[(4)] For all $\alpha>0$, we have
\[
\int_{\mathbb{R}^n}\varphi(x)\exp(-\alpha|x|^2)\,dx \geq 0.
\]
\end{itemize}
\end{exercise}

\begin{solution}
\textbf{Motivation:} Positive definite functions are characterized by Bochner's Theorem as the Fourier transforms of positive measures. This problem explores the basic properties of these functions and their behavior under integration and convolution.

(1) For $k=1$, $c_1 \bar{c}_1 \varphi(0) \geq 0 \implies \varphi(0) \geq 0$.
For $k=2$, let $y_1=y, y_2=0$. The matrix $M = \begin{pmatrix} \varphi(0) & \varphi(y) \\ \varphi(-y) & \varphi(0) \end{pmatrix}$ must be Hermitian and positive semi-definite. Hermitian property implies $\varphi(-y) = \overline{\varphi(y)}$. Positive semi-definiteness implies $\det(M) \geq 0 \implies \varphi(0)^2 - \varphi(y)\varphi(-y) \geq 0 \implies \varphi(0)^2 - |\varphi(y)|^2 \geq 0$, so $|\varphi(y)| \leq \varphi(0)$.

(2) The integral is the limit of Riemann sums of the form $\sum_{i,j} \varphi(y_i-y_j) f(y_i) f(y_j) \Delta y_i \Delta y_j$. Setting $c_i = f(y_i) \Delta y_i$ (which are real since $f$ is nonnegative), the sum is $\sum_{i,j} c_i c_j \varphi(y_i-y_j) \geq 0$ by the definition of positive definiteness.

(3) If $f$ is even, then $(f*f)(x) = \int f(y) f(x-y) dy$.
The integral in (2) is $\iint \varphi(x-y) f(x) f(y) dx dy$. Since $f$ is even, $f(y) = f(-y)$. Let $u = x-y$ and $v = y$. Then $x = u+v$ and the integral becomes:
\[ \iint \varphi(u) f(u+v) f(v) dv du = \int \varphi(u) \left( \int f(u+v) f(v) dv \right) du \]
The inner integral is $\int f(v-(-u)) f(v) dv = (f*f)(-u)$. Since $f$ is even, $f*f$ is even, so $(f*f)(-u) = (f*f)(u)$. Thus, $\int \varphi(u) (f*f)(u) du \geq 0$.

(4) The Gaussian $\exp(-\alpha |x|^2)$ is a positive definite function. Specifically, it can be written as the convolution of a Gaussian with itself: $e^{-\alpha |x|^2} = (g*g)(x)$ for some even function $g$. By part (3), the result follows. Alternatively, by Bochner's theorem, $\varphi$ is the Fourier transform of a positive measure $\mu$, so $\int \varphi(x) e^{-\alpha |x|^2} dx = \int \left( \int e^{i \xi \cdot x} d\mu(\xi) \right) e^{-\alpha |x|^2} dx = \int \left( \int e^{-\alpha |x|^2} e^{i \xi \cdot x} dx \right) d\mu(\xi)$. The inner integral is the Fourier transform of a Gaussian, which is positive.
\end{solution}

\begin{exercise}
For a $2\pi$-periodic function $x\in L^2([0,2\pi])$, let $x_n(t) = x(nt)$, $n=1,2,\ldots$.

Prove that $\{x_n\}$ converges weakly in $L^2([0,2\pi])$ and find the limit.
\end{exercise}

\begin{solution}
\textbf{Motivation:} This problem examines the behavior of fast-oscillating periodic functions. Such functions do not converge strongly (unless constant), but they converge weakly to their average value, as the oscillations "cancel out" against any fixed test function.

Let $x(t) = \sum_{k=-\infty}^{\infty} c_k e^{ikt}$ be the Fourier series of $x$, where $c_k = \frac{1}{2\pi} \int_0^{2\pi} x(t) e^{-ikt} dt$. Then $x_n(t) = x(nt) = \sum_{k} c_k e^{iknt}$.
For any test function $y \in L^2([0, 2\pi])$ with Fourier coefficients $d_m$, the inner product is:
\[ \langle x_n, y \rangle = \int_0^{2\pi} x_n(t) \overline{y(t)} dt = \sum_k \sum_m c_k \overline{d_m} \int_0^{2\pi} e^{iknt} e^{-imt} dt \]
The integral is $2\pi$ if $kn = m$ and $0$ otherwise. Thus:
\[ \langle x_n, y \rangle = 2\pi \sum_k c_k \overline{d_{kn}} \]
As $n \to \infty$, for any fixed $k \neq 0$, the index $kn \to \infty$ (or $-\infty$). Since $y \in L^2$, we have $\lim_{|j|\to\infty} d_j = 0$.
The term for $k=0$ is $2\pi c_0 \overline{d_0}$, which is constant for all $n$.
For the remaining terms, $|\sum_{k \neq 0} c_k \overline{d_{kn}}| \leq (\sum |c_k|^2)^{1/2} (\sum_{k \neq 0} |d_{kn}|^2)^{1/2}$.
Since $\sum |d_j|^2 < \infty$, the tail $\sum_{|j| \geq n} |d_j|^2 \to 0$ as $n \to \infty$. Thus $\sum_{k \neq 0} |d_{kn}|^2 \to 0$.
Therefore, $\lim_{n\to\infty} \langle x_n, y \rangle = 2\pi c_0 \overline{d_0} = 2\pi (\frac{1}{2\pi} \int x) (\frac{1}{2\pi} \int \overline{y}) = \langle \bar{x}, y \rangle$, where $\bar{x} = \frac{1}{2\pi} \int_0^{2\pi} x(t) dt$ is the average of $x$.
The sequence $x_n$ converges weakly to the constant function $c_0 = \frac{1}{2\pi} \int_0^{2\pi} x(t) dt$.
\end{solution}

\begin{exercise}
Let $f$ be an entire function on $\mathbb{C}$ and there exists a constant $C>0$ such that $|f(z)|\leq C\sqrt{|z|}|\cos(z)|$ for all $z\in\mathbb{C}$. Prove that $f$ is identically zero.
\end{exercise}

\begin{solution}
\textbf{Motivation:} This is a problem in complex analysis that combines the use of zeros of holomorphic functions with growth estimates (Liouville's theorem). By dividing by the "growth-driving" term $\cos(z)$, we obtain a simpler function that must be constant due to its slow growth.

The zeros of $\cos(z)$ are $z_k = (k + 1/2)\pi$ for $k \in \mathbb{Z}$. The given inequality implies $f(z_k) = 0$ for all $k$. Since $\cos(z)$ has simple zeros at $z_k$, the function $g(z) = \frac{f(z)}{\cos(z)}$ is an entire function.
The inequality $|f(z)| \leq C \sqrt{|z|} |\cos(z)|$ implies $|g(z)| \leq C \sqrt{|z|}$ for all $z \in \mathbb{C}$.
By the generalized Liouville's theorem, an entire function $g(z)$ satisfying $|g(z)| \leq M |z|^k$ must be a polynomial of degree at most $\lfloor k \rfloor$.
In our case, $k = 1/2$, so $g(z)$ must be a polynomial of degree $0$, i.e., a constant $g(z) = a$.
Thus $f(z) = a \cos(z)$.
Substituting this back into the original inequality: $|a \cos(z)| \leq C \sqrt{|z|} |\cos(z)|$, which implies $|a| \leq C \sqrt{|z|}$ for all $z$ such that $\cos(z) \neq 0$.
Taking the limit as $z \to 0$ (or letting $|z| \to 0$) gives $|a| \leq 0$, hence $a = 0$.
Therefore, $f(z) = 0$ for all $z \in \mathbb{C}$.
\end{solution}

\begin{exercise}
Let $u(t,x)$ satisfy the following equation:
\[
u_t + \sum_{i=1}^n \psi_i(t,x,u)u_{x_i} = \mu\Delta u;\quad u(0,x) = u_0,
\]
where $u_0\in C^2(\mathbb{R}^n)$, $\psi_i$, $i=1,\ldots,n$ are of bounded $C^2$-norm, $\Delta = \sum_{i=1}^n \partial_{x_i}^2$ is the Laplacian in $\mathbb{R}^n$ and $\mu>0$. Assume $|u_0|\leq e^{\Phi/\mu}$ where $\Phi$ is bounded above and has Lipschitz constant $1$. Assume that $\left(\sum_{i=1}^n|\psi_i(t,x,u)|^2\right)^{1/2}\leq A$, where $A$ is a positive constant.

Prove that there exists a constant $C$ depending only on $n$ such that
\[
|u(t,x)| \leq e^{Ct}e^{((A+1)t+\Phi(x)+2)/\mu}, \quad \forall t\geq 0.
\]
\end{exercise}

\begin{solution}
\textbf{Motivation:} This problem uses the probabilistic representation of parabolic PDEs (Feynman-Kac formula) to obtain an exponential bound on the solution. The drift term $\psi$ is handled using the Lipschitz property of the initial data's exponent, and the diffusion part results in a Gaussian-like bound.

We use the Feynman-Kac representation for the solution of the parabolic equation. Let $X_s$ be the diffusion process starting at $X_0 = x$ satisfying the SDE:
\[ dX_s = \psi(t-s, X_s, u) ds + \sqrt{2\mu} dW_s \]
where $W_s$ is a standard $n$-dimensional Brownian motion. The solution $u(t, x)$ can be represented as:
\[ u(t, x) = E[u_0(X_t)] \]
Thus, $|u(t, x)| \leq E[|u_0(X_t)|]$. Using the assumption $|u_0(y)| \leq e^{\Phi(y)/\mu}$, we have:
\[ |u(t, x)| \leq E[e^{\Phi(X_t)/\mu}] \]
Since $\Phi$ has Lipschitz constant $1$, $\Phi(X_t) \leq \Phi(x) + |X_t - x|$. From the SDE:
\[ X_t - x = \int_0^t \psi ds + \sqrt{2\mu} W_t \implies |X_t - x| \leq \int_0^t |\psi| ds + \sqrt{2\mu} |W_t| \leq At + \sqrt{2\mu} |W_t| \]
where we used $|\psi| \leq A$. Thus:
\[ E[e^{\Phi(X_t)/\mu}] \leq e^{\Phi(x)/\mu} e^{At/\mu} E[e^{\sqrt{2\mu}|W_t|/\mu}] = e^{(\Phi(x)+At)/\mu} E[e^{\sqrt{2/\mu}|W_t|}] \]
Let $\lambda = \sqrt{2/\mu}$. We need to estimate $E[e^{\lambda |W_t|}]$. For an $n$-dimensional Brownian motion, $|W_t|$ has the same distribution as $\sqrt{t} \chi_n$, where $\chi_n$ is the chi distribution with $n$ degrees of freedom. It is known that $E[e^{c |W_t|}] \leq C_n e^{c^2 t / 2}$ for some constant $C_n$.
Specifically, $E[e^{\lambda |W_t|}] \leq C_n e^{\lambda^2 t / 2} = C_n e^{(2/\mu) t / 2} = C_n e^{t/\mu}$.
Combining these estimates:
\[ |u(t, x)| \leq C_n e^{t/\mu} e^{(\Phi(x)+At)/\mu} = C_n e^{(\Phi(x) + (A+1)t)/\mu} \]
Taking $C = \ln C_n / t$ (or just bounding $C_n \leq e^{Ct}$ for some $C$), and observing the extra $+2/\mu$ in the exponent provided in the problem allows for the constant $C_n$ and small errors in the Gaussian tail bound. Specifically, the bound $|u(t,x)| \leq e^{Ct} e^{((A+1)t + \Phi(x) + 2)/\mu}$ holds.
\end{solution}
