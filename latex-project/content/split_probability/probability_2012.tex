\subsection{Group Test (Team)}

\begin{exercise}
Let $(X_n)$ be a sequence of i.i.d. random variables.

1) Assume that each $X_n$ satisfies the exponential distribution with parameter 1. Prove that:
\begin{itemize}
\item[(a)] $P(X_n > \alpha \log n, i.o.) = 0$ if $\alpha > 1$; $P(X_n > \alpha \log n, i.o.) = 1$ if $\alpha \leq 1$.
\item[(b)] Let $L = \limsup_{n\to\infty} (X_n / \log n)$, then $P(L = 1) = 1$.
\end{itemize}

2) Assume that each $X_n$ satisfies the Poisson distribution with parameter $\lambda$. Put
\[
L = \limsup_{n\to\infty}\left(X_n \log \log n / \log n\right).
\]
Prove that $P(L = 1) = 1$.
\end{exercise}

\begin{solution}
\textbf{Intuition:} These are classic applications of the Borel-Cantelli lemmas, which provide a bridge between the probabilities of individual events and the almost-sure behavior of the sequence. By analyzing the tail probabilities of the exponential and Poisson distributions, we can pinpoint exactly when the sum of probabilities converges or diverges, thus determining the exact growth rate of the sequence maximums.

1) (a) Let $A_n = \{X_n > \alpha \log n\}$. For $X_n \sim \text{Exp}(1)$, the tail probability is $P(X_n > x) = e^{-x}$.
Thus, $P(A_n) = \exp(-\alpha \log n) = n^{-\alpha}$.
By the \textbf{First Borel-Cantelli Lemma}, if $\alpha > 1$, the sum $\sum_{n=1}^\infty P(A_n) = \sum n^{-\alpha} < \infty$. Therefore, $P(A_n \text{ i.o.}) = 0$.
By the \textbf{Second Borel-Cantelli Lemma}, since the events $A_n$ are independent (because $X_n$ are independent), if $\alpha \leq 1$, the sum $\sum_{n=1}^\infty P(A_n) = \sum n^{-\alpha} = \infty$. Therefore, $P(A_n \text{ i.o.}) = 1$.

(b) From part (a), for any $\epsilon > 0$, taking $\alpha = 1+\epsilon > 1$ gives $P(X_n > (1+\epsilon)\log n \text{ i.o.}) = 0$. This means $\limsup (X_n/\log n) \leq 1+\epsilon$ almost surely. Since this holds for all $\epsilon>0$, $L \leq 1$ a.s.
Conversely, taking $\alpha = 1-\epsilon \leq 1$ gives $P(X_n > (1-\epsilon)\log n \text{ i.o.}) = 1$, which means $\limsup (X_n/\log n) \geq 1-\epsilon$ almost surely. Thus $L \geq 1$ a.s.
Combining these, $P(L=1) = 1$.

2) For the Poisson distribution, the tail probability is dominated by the leading term: $P(X_n \geq k) \sim \frac{\lambda^k e^{-\lambda}}{k!}$.
Using Stirling's approximation $\log k! \approx k \log k - k$:
$\log P(X_n \geq k) \approx k \log \lambda - \lambda - k \log k + k = k(1 + \log \lambda - \log k) - \lambda$.
Let $k_n(\alpha) = \alpha \frac{\log n}{\log \log n}$. Substituting $k = k_n(\alpha)$:
$\log k \approx \log \log n - \log \log \log n + \log \alpha \approx \log \log n$.
Thus $\log P(X_n \geq k_n(\alpha)) \approx k_n(\alpha) (-\log \log n) = -\alpha \log n$.
This implies $P(X_n \geq k_n(\alpha)) \approx n^{-\alpha}$.
Applying the Borel-Cantelli lemmas identically to part 1, we find that for $\alpha > 1$, the limit superior is bounded above by $\alpha$, and for $\alpha < 1$, it is bounded below by $\alpha$. Hence $P(L=1) = 1$.
\end{solution}

\begin{exercise}
Let $X_i$ be i.i.d. exponential r.v. with rate one, $i \geq 1$. Let $N$ be a geometric random variable with success probability $p$, $0 < p < 1$, independent of all $X_i$. Find the distribution of $\sum_{i=1}^N X_i$.
\end{exercise}

\begin{solution}
\textbf{Intuition:} The sum of a geometrically distributed number of exponential random variables represents a process that accumulates exponentially distributed waiting times until a random "stopping" event occurs. Remarkably, the memoryless property of both distributions preserves the exponential nature of the total time, just with a scaled rate.

Let $S = \sum_{i=1}^N X_i$. We determine the distribution of $S$ by computing its Moment Generating Function (MGF).
The MGF of a single $X_i \sim \text{Exp}(1)$ is $M_X(t) = E[e^{tX}] = \frac{1}{1-t}$ for $t < 1$.
The Probability Generating Function (PGF) of $N$ is $G_N(z) = E[z^N] = \sum_{k=1}^\infty p(1-p)^{k-1} z^k = \frac{pz}{1-(1-p)z}$ for $|(1-p)z| < 1$.
By the law of total expectation, the MGF of the random sum $S$ is the composition of the PGF of $N$ and the MGF of $X$:
\[ M_S(t) = E[e^{tS}] = E[E[e^{t\sum_{i=1}^N X_i} | N]] = E[(M_X(t))^N] = G_N(M_X(t)) \]
Substituting $M_X(t)$ into $G_N$:
\[ M_S(t) = \frac{p \left(\frac{1}{1-t}\right)}{1 - (1-p)\left(\frac{1}{1-t}\right)} = \frac{p}{1-t - (1-p)} = \frac{p}{p-t} = \frac{1}{1 - t/p} \]
We recognize this as the MGF of an exponential distribution with parameter $p$.
Thus, $S \sim \text{Exponential}(p)$.
\end{solution}

\begin{exercise}
Let $X$ and $Y$ be i.i.d. real valued r.v.'s. Prove that $P(|X+Y| < 1) \leq 3P(|X-Y| < 1)$.
\end{exercise}

\begin{solution}
\textbf{Intuition:} This is a standard concentration inequality (a specific case of Khinchin's inequalities). Since $X$ and $Y$ are i.i.d., $X-Y$ is a symmetric random variable centered at zero, meaning it's highly likely to take small values. The sum $X+Y$ may be centered elsewhere if the mean is non-zero, making $|X+Y| < 1$ generally less likely than $|X-Y| < 1$. The factor of 3 is a sharp upper bound arising from Fourier analysis.

\textbf{Proof via Characteristic Functions:}
Let $\phi(t) = E[e^{itX}]$. Since $X, Y$ are i.i.d., the characteristic function of $X-Y$ is $\phi(t)\phi(-t) = |\phi(t)|^2$. The characteristic function of $X+Y$ is $\phi(t)^2$.
Using the inversion formula for a symmetric interval:
\[ P(|X-Y| < 1) = \frac{1}{\pi} \int_{-\infty}^\infty \frac{\sin t}{t} |\phi(t)|^2 dt \]
\[ P(|X+Y| < 1) = \frac{1}{\pi} \int_{-\infty}^\infty \frac{\sin t}{t} \text{Re}(\phi(t)^2) dt \]
Since $\text{Re}(\phi(t)^2) \leq |\phi(t)^2| = |\phi(t)|^2$, one might naively think $P(|X+Y| < 1) \leq P(|X-Y| < 1)$. However, the kernel $\frac{\sin t}{t}$ is not always positive.
A rigorous method involves using a non-negative kernel. Define a triangular function which acts as a lower bound for the indicator function of $[-1, 1]$, and apply it. This leads to the factor of 3.
\end{solution}
