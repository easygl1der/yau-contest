\begin{exercise}
\label{geometry:2010:1}
Let $D^{*} = \{ (x, y) \in \mathbb{R}^{2} \mid 0 < x^{2} + y^{2} < 1 \}$ be the punctured unit disc in the Euclidean plane. Let $g$ be the complete Riemannian metric on $D^{*}$ with constant curvature $-1$. Find the distance under the metric between the points $(e^{-2\pi}, 0)$ and $(-e^{-\pi}, 0)$.
\end{exercise}

\begin{solution}
The complete Riemannian metric of constant curvature $-1$ on the punctured disc $D^*$ is the Poincaré metric for a cusp. It can be obtained from the upper half-plane $\mathbb{H} = \{ z = x + iy \mid y > 0 \}$ by the universal covering map $\pi: \mathbb{H} \to D^*$, $\pi(z) = e^{iz}$.
The hyperbolic metric on $\mathbb{H}$ is $ds^2 = \frac{dx^2 + dy^2}{y^2}$.
The covering map $\pi(x + iy) = e^{i(x + iy)} = e^{-y} e^{ix}$.
In polar coordinates $(r, \theta)$ on $D^*$, we have $r = e^{-y}$ and $\theta = x$.
Then $y = -\log r$ and $dy = -\frac{dr}{r}$, $dx = d\theta$.
The metric $g$ on $D^*$ is:
\[ g = \frac{d\theta^2 + (dr/r)^2}{(-\log r)^2} = \frac{dr^2 + r^2 d\theta^2}{(r \log r)^2}. \]
We want to find the distance between $p = (e^{-2\pi}, 0)$ and $q = (-e^{-\pi}, 0)$.
In polar coordinates: $p$ is $(r_p, \theta_p) = (e^{-2\pi}, 0)$, and $q$ is $(r_q, \theta_q) = (e^{-\pi}, \pi)$.
To find the distance in $D^*$, we lift these points to $\mathbb{H}$.
A lift of $p$ is $z_p = 0 + i(2\pi)$.
A lift of $q$ is $z_q = \pi + i(\pi)$.
The distance between two points $z_1, z_2$ in $\mathbb{H}$ is:
\[ d(z_1, z_2) = \operatorname{arccosh}\left( 1 + \frac{|z_1 - z_2|^2}{2 \operatorname{Im}(z_1) \operatorname{Im}(z_2)} \right). \]
Here, $|z_p - z_q|^2 = |\pi + i(\pi - 2\pi)|^2 = |\pi - i\pi|^2 = 2\pi^2$.
$\operatorname{Im}(z_p) = 2\pi$, $\operatorname{Im}(z_q) = \pi$.
\[ d = \operatorname{arccosh}\left( 1 + \frac{2\pi^2}{2(2\pi)(\pi)} \right) = \operatorname{arccosh}\left( 1 + \frac{2\pi^2}{4\pi^2} \right) = \operatorname{arccosh}(1.5). \]
Using $\operatorname{arccosh}(x) = \log(x + \sqrt{x^2 - 1})$:
\[ d = \log(1.5 + \sqrt{2.25 - 1}) = \log(1.5 + \sqrt{1.25}) = \log\left(\frac{3 + \sqrt{5}}{2}\right). \]
Since $\left(\frac{1+\sqrt{5}}{2}\right)^2 = \frac{6+2\sqrt{5}}{4} = \frac{3+\sqrt{5}}{2}$, we have:
\[ d = 2 \log\left(\frac{1+\sqrt{5}}{2}\right). \]
This is the hyperbolic distance between the lifted points. Since the metric on $D^*$ is the quotient metric, the distance in $D^*$ is the minimum over all lifts. However, for these points, the shortest path is the one lifted.
\end{solution}

\begin{exercise}
\label{geometry:2010:2}
Show that every closed hypersurface in $\mathbb{R}^{n}$ has a point at which the second fundamental form is positive definite.
\end{exercise}

\begin{solution}
Let $M \subset \mathbb{R}^n$ be a closed (compact without boundary) hypersurface.
Since $M$ is compact, there exists a point $p \in M$ that is furthest from the origin.
Let $R = |p|$. Then $M$ is contained in the closed ball $\bar{B}_R(0)$, and $M$ touches the sphere $S_R(0) = \partial B_R(0)$ at $p$.
At the point $p$, the tangent space $T_p M$ must coincide with the tangent space $T_p S_R(0)$.
Let $N$ be the unit normal to $M$ at $p$ pointing outward from the origin.
For any unit vector $v \in T_p M$, let $\gamma(s)$ be a geodesic in $M$ with $\gamma(0) = p$ and $\dot{\gamma}(0) = v$.
Since $M$ is inside the ball, the curve $\gamma(s)$ is inside the ball, so $|\gamma(s)|^2 \leq R^2$.
Differentiating $|\gamma(s)|^2$ at $s=0$:
$\frac{d}{ds} |\gamma(s)|^2 = 2 \langle \gamma(s), \dot{\gamma}(s) \rangle$. At $s=0$, $2 \langle p, v \rangle = 0$ (since $p$ is normal to the sphere).
$\frac{d^2}{ds^2} |\gamma(s)|^2 = 2 \langle \dot{\gamma}(s), \dot{\gamma}(s) \rangle + 2 \langle \gamma(s), \ddot{\gamma}(s) \rangle$.
At $s=0$: $2|v|^2 + 2 \langle p, \ddot{\gamma}(0) \rangle \leq 0$.
Since $|v|=1$, $1 + \langle p, \ddot{\gamma}(0) \rangle \leq 0$, so $\langle p, \ddot{\gamma}(0) \rangle \leq -1$.
The acceleration $\ddot{\gamma}(0)$ of the geodesic in $M$ is related to the second fundamental form $II$ by $\ddot{\gamma}(0) = II(v, v) N$.
Substituting this: $\langle p, II(v, v) N \rangle \leq -1$.
Since $N$ is the unit outward normal, $p = RN$.
So $R \, II(v, v) \leq -1$, which means $II(v, v) \leq -1/R$.
This shows that all principal curvatures (relative to the outward normal) are negative.
If we choose the normal pointing inward, the second fundamental form is positive definite.
Thus, at the point furthest from the origin, the second fundamental form is definite (positive or negative depending on orientation).
\end{solution}

\begin{exercise}
\label{geometry:2010:3}
Prove that the real projective space $\mathbb{RP}^{n}$ is orientable if and only if $n$ is odd.
\end{exercise}

\begin{solution}
Consider the double covering $\pi: S^n \to \mathbb{RP}^n$. The manifold $\mathbb{RP}^n$ is orientable if and only if the deck transformation $A: S^n \to S^n$ given by $A(x) = -x$ (the antipodal map) is orientation-preserving.
The antipodal map $A$ on $S^n \subset \mathbb{R}^{n+1}$ is the restriction of the linear map $-I: \mathbb{R}^{n+1} \to \mathbb{R}^{n+1}$.
The determinant of $-I$ on $\mathbb{R}^{n+1}$ is $(-1)^{n+1}$.
An $n+1$-dimensional linear map $L$ is orientation-preserving if $\det L > 0$.
However, we care about the effect on the $n$-form (volume form) on the sphere.
The standard volume form on $S^n$ is $\omega = \sum_{i=1}^{n+1} (-1)^{i-1} x_i dx_1 \wedge \dots \wedge \widehat{dx_i} \wedge \dots \wedge dx_{n+1}$.
Pulling back by $A$: $A^* x_i = -x_i$ and $A^* dx_i = -dx_i$.
In the expression for $\omega$, each term has one $x_i$ and $n$ differentials.
So $A^* \omega = (-1) \cdot (-1)^n \omega = (-1)^{n+1} \omega$.
For $A$ to preserve orientation, we need $A^* \omega = \omega$, so $(-1)^{n+1} = 1$.
This requires $n+1$ to be even, which means $n$ must be odd.
Thus $\mathbb{RP}^n$ is orientable if and only if $n$ is odd.
\end{solution}

\begin{exercise}
\label{geometry:2010:4}
Suppose $\pi : M_{1} \to M_{2}$ is a $C^{\infty}$ map of one connected differentiable manifold to another. Suppose for each $p \in M_{1}$, the differential $\pi_{*} : T_{p} M_{1} \to T_{\pi(p)} M_{2}$ is a vector space isomorphism.
(a) Show that if $M_{1}$ is connected and $\pi$ is proper, then $\pi$ is a covering space projection.
(b) Give an example where $M_{2}$ is compact but $\pi : M_{1} \to M_{2}$ is not a covering space.
\end{exercise}

\begin{solution}
(a) Since $\pi_*$ is an isomorphism at every point, $\pi$ is a local diffeomorphism by the Inverse Function Theorem. This implies $\pi(M_1)$ is an open set in $M_2$. Since $M_1$ is compact and $\pi$ is continuous, $\pi(M_1)$ is also a closed set in $M_2$. Since $M_2$ is connected, $\pi$ is surjective.
A proper local diffeomorphism between manifolds is a covering map. To see this, let $q \in M_2$. Its preimage $\pi^{-1}(q)$ is a discrete set (since $\pi$ is a local diffeomorphism). Since $\pi$ is proper, $\pi^{-1}(q)$ is compact, thus it is a finite set $\{p_1, \dots, p_k\}$.
For each $p_i$, there exist disjoint open neighborhoods $U_i$ such that $\pi|_{U_i}: U_i \to V_i$ is a diffeomorphism. Let $V = (\cap V_i) \setminus \pi(M_1 \setminus \cup U_i)$. Since $\pi$ is proper, it is a closed map, so $\pi(M_1 \setminus \cup U_i)$ is closed. Thus $V$ is an open neighborhood of $q$ that is evenly covered by $\pi$.

(b) Let $M_1 = (0, 1)$ and $M_2 = S^1$. Define $\pi(t) = e^{2\pi i t}$.
For every $t \in (0, 1)$, $\pi'(t) = 2\pi i e^{2\pi i t} \neq 0$, so $\pi_*$ is an isomorphism.
However, $\pi$ is not a covering map because it is not surjective (it misses the point $1 \in S^1$). Even if we made it surjective (e.g. $\pi: (0, 2) \to S^1$), it wouldn't be a covering if some points are not "evenly covered" (the point $1$ would have two preimages $1/2$ and $3/2$, but small neighborhoods wouldn't work at the boundaries).
Actually, the simplest example is $\pi: (0, 1) \to S^1$. It is a local diffeomorphism but not a covering map.
\end{solution}

\begin{exercise}
\label{geometry:2010:5}
Let $\Sigma_{g}$ be the closed orientable surface of genus $g$. Show that if $g > 1$, then $\Sigma_{g}$ is a covering space of $\Sigma_{2}$.
\end{exercise}

\begin{solution}
A closed orientable surface $\Sigma_g$ covers $\Sigma_h$ if and only if there exists a $k$-sheeted covering such that the Euler characteristics satisfy $\chi(\Sigma_g) = k \chi(\Sigma_h)$.
We have $\chi(\Sigma_g) = 2 - 2g$ and $\chi(\Sigma_2) = 2 - 2(2) = -2$.
The condition is $2 - 2g = k(-2)$, which simplifies to $1 - g = -k$, or $k = g - 1$.
Since $g > 1$, $k = g - 1 \geq 1$ is a positive integer.
To construct the covering: Represent $\Sigma_2$ as a $4$-gon with identifying edges, or more easily, as two tori $T^2$ joined by a neck.
One can use the fact that $\Sigma_2$ is a quotient of the upper half plane by a Fuchsian group $\Gamma$. Any subgroup of $\Gamma$ of index $k$ corresponds to a $k$-sheeted covering. Such subgroups always exist for any $k \geq 1$.
Specifically, the fundamental group $\pi_1(\Sigma_2)$ has a surjection onto $\mathbb{Z}$. The kernel of the map $\pi_1(\Sigma_2) \to \mathbb{Z} \to \mathbb{Z}_k$ is a subgroup of index $k$, corresponding to a $k$-sheeted covering of genus $g = k(2-1) + 1 = k+1$.
Setting $k = g-1$ gives a covering of $\Sigma_2$ by $\Sigma_g$.
\end{solution}

\begin{exercise}
\label{geometry:2010:6}
Let $M$ be a smooth 4-dimensional manifold. A symplectic form is a closed 2-form $\omega$ on $M$ such that $\omega \wedge \omega$ is a nowhere vanishing 4-form.
(a) Construct a symplectic form on $\mathbb{R}^{4}$.
(b) Show that there are no symplectic forms on $S^{4}$.
\end{exercise}

\begin{solution}
(a) On $\mathbb{R}^4$ with coordinates $(x_1, y_1, x_2, y_2)$, define:
\[ \omega = dx_1 \wedge dy_1 + dx_2 \wedge dy_2. \]
Then $d\omega = 0$ (closed).
$\omega \wedge \omega = (dx_1 \wedge dy_1 + dx_2 \wedge dy_2) \wedge (dx_1 \wedge dy_1 + dx_2 \wedge dy_2)$
$= dx_1 \wedge dy_1 \wedge dx_2 \wedge dy_2 + dx_2 \wedge dy_2 \wedge dx_1 \wedge dy_1 = 2 dx_1 \wedge dy_1 \wedge dx_2 \wedge dy_2$.
This is a nowhere-vanishing 4-form (volume form).

(b) Suppose $S^4$ has a symplectic form $\omega$.
Since $H^2(S^4; \mathbb{R}) = 0$, every closed 2-form is exact.
Thus $\omega = d\eta$ for some 1-form $\eta$.
Then $\omega \wedge \omega = d\eta \wedge \omega = d(\eta \wedge \omega)$.
Integrating over $S^4$ and using Stokes' theorem:
\[ \int_{S^4} \omega \wedge \omega = \int_{S^4} d(\eta \wedge \omega) = \int_{\partial S^4} \eta \wedge \omega = 0. \]
However, $\omega \wedge \omega$ is a nowhere-vanishing 4-form. On a compact oriented manifold, the integral of a nowhere-vanishing top-form must be non-zero (it's a volume form).
This is a contradiction. Therefore, $S^4$ admits no symplectic form.
\end{solution}

\begin{exercise}
\label{geometry:2010:7}
Prove that the stereographic projection $\Pi: S^n \setminus \{N\} \to \mathbb{R}^n$ is a conformal map, and derive the standard metric of $S^n$ in stereographic coordinates.
\end{exercise}

\begin{solution}
The stereographic projection from $N=(0, \dots, 1)$ is $\Pi(x) = \frac{x}{1-x_{n+1}}$ for $x \in S^n$.
The inverse is $\phi(u) = \left( \frac{2u}{1+|u|^2}, \frac{|u|^2-1}{1+|u|^2} \right)$.
The standard metric on $S^n$ is $g_{S^n} = \sum_{i=1}^{n+1} dx_i^2$.
Differentiating $\phi(u)$:
\[ dx_i = \frac{2 du_i}{1+|u|^2} - \frac{4 u_i (u \cdot du)}{(1+|u|^2)^2}, \quad dx_{n+1} = \frac{4 u \cdot du}{(1+|u|^2)^2}. \]
Summing the squares:
$\sum_{i=1}^n dx_i^2 = \frac{4 |du|^2}{(1+|u|^2)^2} - \frac{16 (u \cdot du)^2}{(1+|u|^2)^3} + \frac{16 |u|^2 (u \cdot du)^2}{(1+|u|^2)^4}$.
Adding $dx_{n+1}^2 = \frac{16 (u \cdot du)^2}{(1+|u|^2)^4}$:
\[ \sum_{i=1}^{n+1} dx_i^2 = \frac{4 |du|^2}{(1+|u|^2)^2} + \frac{16 (u \cdot du)^2}{(1+|u|^2)^4} (1 + |u|^2) - \frac{16 (u \cdot du)^2}{(1+|u|^2)^3} = \frac{4 |du|^2}{(1+|u|^2)^2}. \]
The metric is $g = \frac{4}{(1+|u|^2)^2} \sum du_i^2$. This is a scalar multiple of the Euclidean metric, so it is conformal.
\end{solution}

\begin{exercise}
\label{geometry:2010:8}
Let $M$ be a (connected) Riemannian manifold of dimension 2. Let $f$ be a smooth non-constant function on $M$ such that $f$ is bounded from above and $\Delta f \geq 0$ everywhere on $M$. Show that there does not exist any point $p \in M$ such that $f(p) = \sup \{f(x) : x \in M\}$.
\end{exercise}

\begin{solution}
The condition $\Delta f \geq 0$ means $f$ is subharmonic (using the analyst's convention where $\Delta = \sum \partial_i^2$). In geometry, $\Delta$ is often defined as $\text{div} \nabla$, which is the same.
The Strong Maximum Principle states that if a subharmonic function achieves its maximum in the interior of its domain, then the function must be constant.
Suppose there exists $p \in M$ such that $f(p) = \sup f$. Let $M_{max} = \{ x \in M \mid f(x) = f(p) \}$.
$M_{max}$ is non-empty and closed (by continuity of $f$).
By the Strong Maximum Principle, for any point $q \in M_{max}$, there is a neighborhood of $q$ contained in $M_{max}$. Thus $M_{max}$ is also open.
Since $M$ is connected, $M_{max} = M$. This means $f$ is constant, which contradicts the assumption that $f$ is non-constant.
Therefore, no such point $p$ exists.
\end{solution}

\begin{exercise}
\label{geometry:2010:9}
Let $M$ be a compact smooth manifold of dimension $d$. Prove that there exists some $n \in \mathbb{Z}^{+}$ such that $M$ can be regularly embedded in the Euclidean space $\mathbb{R}^{n}$.
\end{exercise}

\begin{solution}
This is the Whitney Embedding Theorem for compact manifolds.
Since $M$ is compact, it can be covered by finitely many coordinate charts $(U_i, \phi_i)$, $i=1, \dots, k$.
Let $\{\rho_i\}$ be a partition of unity subordinate to this cover.
Define $\psi_i: M \to \mathbb{R}^d$ by $\psi_i(x) = \rho_i(x) \phi_i(x)$ (extended by 0 outside $U_i$).
Define a map $f: M \to \mathbb{R}^{(d+1)k}$ by:
\[ f(x) = (\psi_1(x), \dots, \psi_k(x), \rho_1(x), \dots, \rho_k(x)). \]
This map is smooth.
Injectivity: If $f(x) = f(y)$, then $\rho_i(x) = \rho_i(y)$ for all $i$. For at least one $i$, $\rho_i(x) > 0$. Then $\phi_i(x) = \psi_i(x)/\rho_i(x) = \psi_i(y)/\rho_i(y) = \phi_i(y)$. Since $\phi_i$ is a diffeomorphism on $U_i$, $x=y$.
Immersion: At any $x$, there is $i$ such that $\rho_i(x) > 0$. The differential of $(\psi_i, \rho_i)$ at $x$ has rank $d$ because $\phi_i$ is a coordinate map.
Since $M$ is compact, an injective immersion is a regular embedding.
\end{solution}

\begin{exercise}
\label{geometry:2010:10}
Show that any $C^{\infty}$ function $f$ on a compact smooth manifold $M$ (without boundary) must have at least two critical points. When $M$ is the 2-torus, show that $f$ must have more than two critical points.
\end{exercise}

\begin{solution}
A critical point of $f$ is a point where the differential $df$ vanishes.
Since $M$ is compact, $f$ attains its global maximum and global minimum.
At a maximum or minimum point, the derivative must be zero in all directions, so these are critical points.
If $f$ is not constant, the maximum and minimum values are distinct, so the points are distinct.
If $f$ is constant, every point is a critical point. In both cases, there are at least two.

For $M = T^2 = S^1 \times S^1$, Morse theory states that for a Morse function (one with non-degenerate critical points), the number of critical points of index $k$ is at least the Betti number $b_k$.
For $T^2$, $b_0 = 1, b_1 = 2, b_2 = 1$.
The total number of critical points is at least $1 + 2 + 1 = 4$.
For a general smooth function, the Lusternik-Schnirelmann category $\text{cat}(T^2) = 3$. The number of critical points is at least $\text{cat}(M) = 3$.
Actually, on the torus, any smooth function must have at least 3 critical points, and if they are non-degenerate, at least 4.
Thus $f$ has more than two critical points.
\end{solution}

\begin{exercise}
\label{geometry:2010:11}
Construct a space $X$ with $H_{0}(X) = \mathbb{Z}$, $H_{1}(X) = \mathbb{Z}_{2} \times \mathbb{Z}_{3}$, $H_{2}(X) = \mathbb{Z}$, and all other homology groups of $X$ vanishing.
\end{exercise}

\begin{solution}
We can construct $X$ by taking the wedge sum of spaces that contribute to each homology group.
1. For $H_1 = \mathbb{Z}_2 \oplus \mathbb{Z}_3$: Let $M_2$ be the Moore space $M(\mathbb{Z}_2, 1)$, which is the real projective plane $\mathbb{RP}^2$ (or a circle with a 2-cell attached by a map of degree 2). It has $H_1 = \mathbb{Z}_2$ and $H_2 = 0$.
2. Let $M_3$ be the Moore space $M(\mathbb{Z}_3, 1)$, which is a circle with a 2-cell attached by a map of degree 3. It has $H_1 = \mathbb{Z}_3$ and $H_2 = 0$.
3. For $H_2 = \mathbb{Z}$: Let $S^2$ be the 2-sphere. It has $H_1 = 0$ and $H_2 = \mathbb{Z}$.
Define $X = \mathbb{RP}^2 \vee M_3 \vee S^2$.
Using the property $\tilde{H}_n(A \vee B) = \tilde{H}_n(A) \oplus \tilde{H}_n(B)$:
$H_0(X) = \mathbb{Z}$ (since $X$ is connected).
$H_1(X) = H_1(\mathbb{RP}^2) \oplus H_1(M_3) \oplus H_1(S^2) = \mathbb{Z}_2 \oplus \mathbb{Z}_3 \oplus 0 = \mathbb{Z}_2 \times \mathbb{Z}_3$.
$H_2(X) = H_2(\mathbb{RP}^2) \oplus H_2(M_3) \oplus H_2(S^2) = 0 \oplus 0 \oplus \mathbb{Z} = \mathbb{Z}$.
Higher homology groups are zero.
\end{solution}

\begin{exercise}
\label{geometry:2010:12}
(a) Define the degree $\deg f$ of a $C^{\infty}$ map $f : S^{2} \to S^{2}$ and prove that it is well-defined.
(b) Prove in detail that for each integer $k \in \mathbb{Z}$, there is a $C^{\infty}$ map $f : S^{2} \to S^{2}$ of degree $k$.
\end{exercise}

\begin{solution}
(a) Let $f: S^2 \to S^2$ be smooth. The degree $\deg f$ is the integer $k$ such that for any $\omega \in \Omega^2(S^2)$, $\int_{S^2} f^* \omega = k \int_{S^2} \omega$.
To show it is well-defined: $H^2_{de R}(S^2) \cong \mathbb{R}$. Any two volume forms $\omega_1, \omega_2$ with same integral differ by an exact form $d\eta$. Since $f^*(d\eta) = d(f^*\eta)$, the integral of $f^* d\eta$ is zero by Stokes. Thus the ratio $k$ is independent of the choice of $\omega$.
Alternatively, $\deg f$ is the number of points in $f^{-1}(y)$ for a regular value $y$, counted with sign of the Jacobian.

(b) Identify $S^2$ with $\mathbb{C} \cup \{\infty\}$.
For $k \geq 0$, let $f(z) = z^k$. This map is smooth. At any point $z \neq 0, \infty$, it has $k$ preimages (roots of unity). The Jacobian is $|f'(z)|^2 = |k z^{k-1}|^2 > 0$, so all preimages have positive sign. Thus $\deg f = k$.
For $k < 0$, let $f(z) = (\bar{z})^{|k|}$. This is the composition of the degree $|k|$ map with the reflection $z \mapsto \bar{z}$, which has degree $-1$. Thus the degree is $-|k| = k$.
\end{solution}
